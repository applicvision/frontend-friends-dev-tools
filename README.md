# Frontend Friends Dev Tools

This repository contains developer tools to enhance the experience of using the [Frontend Friends](https://github.com/applicvision/frontend-friends) framework.

It consists of two main projects. For more info, please consult the Readme files in the respective projects:

## Frontend Friends Diagnostics [→](./frontend-friends-diagnostics)
An NPM package containing a CLI tool and an API to analyze and validate HTML and interpolations in Frontend Friends tagged templates.

## Visual Studio Code Extension [→](./vscode-extension)
A Visual Studio Code extension that provides syntax highlighting, rich IntelliSense, and real-time diagnostics directly in your editor.
